# To-do List
Aplicativo em Flutter (versão: 2.2.3) onde você pode organizar sua lista de afazeres.

## Requisitos mínimos
- Visulizar tarefas salvas.
- Completar uma tarefa.
- Salvar nova tarefa.
- Deletar tarefa salva.
